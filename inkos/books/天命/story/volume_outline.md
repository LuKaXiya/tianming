- 冲突：房东说"三天没交租，给我滚"
- 反击：陆尘把凉拖甩到房东脸上

## 黄金三章详解

### 第1章：34D大小姐

**开头钩子**（第一段必须抓住读者）
```
"mlgbd！"

陆尘把行李箱甩在脚边，站在城中村巷口，抬头看了眼天——热得他想把太阳射下来。

手机又响了。他看了眼来电显示，眉头皱得能夹死苍蝇。

"喂，陆先生，您这个月的贷款……"

"知道了知道了，下个月一定还。"

挂了电话，他看了眼短信：欠款余额 31,247.36 元。
```

**核心冲突**
- 被房东赶出来
- 欠债3万+
- 无处可去

**金手指伏笔**
- 提到"师父教剑的日子"（暗示有背景）
- 捡到一张"快递单"（引出林婉儿）

**结尾钩子**
```
陆尘拖行李走过天桥底，看见一张贴在电线杆上的传单——

【全息游戏"天命"公测在即！戴上头盔，改变命运！】

他嗤笑一声。

"改变命运？老子命里缺的是钱。"

然后他继续拖行李，没注意到身后有一辆保时捷缓缓停下，车窗摇下，一双34D的眼睛正看着他。
```

### 第2章：剑心

**第二章任务：展示金手指，让读者看到爽点预期**

**开头承接**
```
林婉儿在天桥底下找到了那个男人。

准确地说，是那个男人睡在天桥底下，姿势销魂，嘴角还挂着口水。

"喂。"

陆尘睁开眼，看见一张精致到不像真人的脸。34D，没化妆，穿着一件看不出牌子的白T恤，但就是好看得让人想把眼睛黏上去。

"看够了？"她皱眉。
```

**金手指展示**
```
陆尘正要说话，突然看见林婉儿手里拿着的——

一张传单。

和他在天桥底下看到的一模一样。

"你也玩这个游戏？"陆尘脱口而出。

林婉儿微微一愣，打量了他几眼："你知道天命？"

"废话，全息游戏元年……"

他突然停住。

因为他发现自己脑子里突然多了一些东西。

不是记忆，是……感觉。

一种很熟悉的感觉。

像是有人在告诉他：这张传单，很重要。
```

**爽点预期**
```
林婉儿递给他一张名片。

"明天上午十点，宁海大学北门，我给你介绍个工作。"

然后她上车走了。

陆尘看了眼名片——

【天海集团·林婉儿】

"我靠……"

他低头看了眼自己的穷酸样，又看了眼那张名片。

"34D大小姐？给我介绍工作？"

他抬头看了眼天。

"太阳打西边出来了？"
```

**结尾钩子**
```
当天晚上，陆尘做了一个梦。

梦里，一个穿白衣的老头站在悬崖边，手持长剑。

"剑心通明，是为大道。"

"徒儿，记住——"

"剑，可斩天。"

陆尘猛地惊醒，发现自己躺在天桥底下，而天桥上方的广告牌，正在播放天命的预告片。

全服公测倒计时：72小时。
```

### 第3章：剑起波澜

**第三章任务：确立第一个具体可达成的目标**

**开头承接**
```
天命公测日。

陆尘坐在城中村网吧角落，戴着头盔，浑身只穿了一条大裤衩。

旁边的大叔看了眼他那张穷酸脸，嗤笑："又一个做白日梦的。"

陆尘没理他，深吸一口气。

"进入游戏。"

眼前一黑。

再睁开眼——

【欢迎来到天命·新手村·桃花村】

他站在一片桃花林中，粉色的花瓣飘落，远处是茅草屋顶，炊烟袅袅。

其他新手玩家也在四处张望，有人兴奋，有人茫然，有人已经开始抢夺地上的新手装备。

陆尘却愣在原地。

因为他发现了不对劲。

这游戏……

太真实了。
```

**确立第一个目标**
```
新手任务提示音在他脑海里响起：

【主线任务：击杀10只野猪】
【奖励：100经验，10铜币】
【提示：新手装备可在铁匠铺领取】

陆尘没有急着去领装备。

他在观察。

因为他发现，这个游戏的战斗系统——

不需要按键。

不需要鼠标。

只需要……想象。

"突刺。"

他的新手木剑刺出，前方的空气仿佛被划破。

虽然没有伤害，但那种感觉——

太熟悉了。

像是师父当年教他的第一招。
```

**爽点落地**
```
一个时辰后。

【系统提示：恭喜您击杀野猪王！】

全服公告：

【首杀成就达成：野猪王】
【奖励：经验+500，铜币+50，黑铁器·桃木剑】
【全服广播：恭喜玩家"逍遥自在"达成首杀成就！】

周围的玩家全都看过来。

"卧槽，首杀！"
"这人是肝帝吧？才开服一小时！"
"黑铁器！我擦我也要！"

陆尘看着手里的桃木剑，剑身上隐约有光芒流转。

他嘴角一勾。

"师父，你教的剑，果然能杀人。"

然后他看向全服广播上的名字——逍遥自在。

"从今天起，这游戏里只有一个传说。"

"叫陆尘。"
```

**结尾钩子**
```
就在这时，一道清冷的声音在他身后响起：

"你的剑……跟谁学的？"

陆尘转身，看见一个NPC。

准确地说，是一个美得不像NPC的NPC。

她穿着白色长裙，手持长剑，眼神清冷如霜。

头顶上写着三个字：流霜。

而陆尘手里的桃木剑，正在微微颤动。

像是在回应什么。
```
